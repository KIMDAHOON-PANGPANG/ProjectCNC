단검 마커 액션/프로토타이핑 로드맵

카타리나 W 스킬을 횡스크롤 닌자 액션으로 가져온 코어 메커닉 검증 플랜. 6단계 게이팅으로 진행하며 각 단계마다 Kill Criteria를 두고 재미가 나오지 않으면 즉시 피벗·드롭 결정합니다.

Engine
Godot 4.6 · GDScript
Reference
Cyber Shadow · 닌자일섬 · The Messenger
Total ETA
10 ~ 14주 (Vertical Slice 까지)
Team Size
Solo · 다훈 (combat/code)
▼ CORE HYPOTHESIS

플레이어는 "단검을 어디에 박을지 결정하는 1초"의 무게 때문에 계속 플레이한다. 단검 = 미래의 내 위치를 예약하는 공간 약속(spatial commitment)이고, 이 약속을 전투 + 플랫포밍 양쪽에서 동일한 동사로 푸는 것이 차별점이다.

OVERALL PROGRESS 0 / 0

Phase 5 종료 후 / Go-No-Go 분기

Vertical Slice 외부 플레이테스트(5명)와 본인 30분 풀플레이를 끝낸 다음, 아래 세 갈래 중 하나로 결정합니다. 직감으로 가지 말고 데이터로 가세요.

GO
5명 중 3명 이상 "또 하고 싶다" + 본인 30분 플레이에 매너리즘 없음
  • 풀 프로덕션 일정 산정 (1챕터 × 8~10)
  • 아트/사운드 외주 또는 에셋 파이프라인 본격화
  • 스팀 페이지 오픈 + 위시리스트 수집 시작
PIVOT
메커닉은 살아있으나 장르/스코프 / 톤이 안 맞음
  • 로그라이트 단편 / 보스러시 모드로 전환 검토
  • 스코어어택(닌자일섬 형) vs 메트로배니아(Cyber Shadow 형) 재선택
  • 코어 메커닉 유지, 셸만 교체 → 시간 손실 최소화
KILL
5룸 중 3룸 이상이 메커닉 없이 풀려버림 = 메커닉 자체가 약함
  • 코어 가설 폐기, 다른 메커닉으로 재시작
  • 여기까지의 자산은 다음 프로젝트 토대로 보관
  • Building Breaker 등 다른 트랙으로 전환
